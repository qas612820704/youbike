# youbike

## Installation

```bash
  $ pip install -r requirements.txt
  # or
  $ make install
```

## Configuration

Please edit `SETTINGS.PY`

- `YOUBIKE_DATA_URL` *(default = 'http://data.taipei/youbike')*
- `SAVE_FILE_DIR` *(default = os.path.join(BASE_DIR, 'UB_DATA'))*

## Usage

``` bash
  $ make download
```
